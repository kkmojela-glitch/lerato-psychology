# Lerato Psychology Website

A modern, responsive website built for a psychology practice. The website is designed to create a calm, professional online presence while making it easy for clients to learn about the practice and request appointments.

## Features

- Responsive design for desktop, tablet, and mobile
- Modern and clean user interface
- About section
- Services section
- Testimonials
- Contact information
- Request Appointment section
- Smooth scrolling navigation

## Technologies Used

- HTML5
- CSS3
- JavaScript
